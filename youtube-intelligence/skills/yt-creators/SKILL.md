# Yt Creators

Manage trusted YouTube creator profiles. View, add, update, and remove

## Usage

```
/youtube-intelligence:yt-creators                   # List all creators
/youtube-intelligence:yt-creators add channel:"Matt Wolfe" trust:"medium"
```
