# Idea Matrix

Quick comparative scoring matrix for 2-5 business ideas side-by-side.

## Usage

```
/business-idea-analyzer:idea-matrix "AI photo editor" vs "AI video editor"
/business-idea-analyzer:idea-matrix "Shopify analytics" vs "eBay listing tool"
```

## When to Use

- When you need to invoke /business-idea-analyzer:idea-matrix
- When the user's request matches the trigger keywords above
