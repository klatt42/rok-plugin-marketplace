# Idea Shortlist

Recall, filter, and re-rank the most recent idea finder shortlist.

## Usage

```
/business-idea-finder:idea-shortlist                # Show full list
/business-idea-finder:idea-shortlist --tier=HOT     # Filter to HOT tier
```

## When to Use

- When you need to invoke /business-idea-finder:idea-shortlist
- When the user's request matches the trigger keywords above
