# Seo Report

Generate export reports from SEO audit data collected in the current session.

## Usage

```
/seo-content-optimizer:seo-report                   # Export all formats
/seo-content-optimizer:seo-report format:pdf
```

## When to Use

- When you need to invoke /seo-content-optimizer:seo-report
- When the user's request matches the trigger keywords above
